My Website Design System

A comprehensive guide to the visual vocabulary and components of our website.

Color Palette

Primary Blue

Hex: #007bff

Use: Key branding elements, primary buttons, main links, and accents.

Secondary Teal

Hex: #17a2b8

Use: Complementary actions, secondary buttons, and decorative elements.

Success Green

Hex: #28a745

Use: Positive feedback, form success messages, and "complete" states.

Danger Red

Hex: #dc3545

Use: Error messages, destructive actions (e.g., delete), and warnings.

Light Gray

Hex: #f8f9fa

Use: Subtle background for sections, alternating table rows, and borders.

Dark Gray

Hex: #343a40

Use: Primary text, headings, and strong content elements.

Specific Text Styles

These definitions ensure consistent application of typography across the site.

Main Heading (H1)

font-family: var(--font-heading);
font-weight: 700;
font-size: 3.5rem;
color: var(--color-dark-gray);

H1: Website Main Title Example

Subheading (H2)

font-family: var(--font-heading);
font-weight: 400;
font-size: 2.0rem;
color: var(--color-dark-gray);

H2: Section Subheading Title

Body Text (Paragraph)

font-family: var(--font-primary);
font-weight: 400;
font-size: 1.1rem;
line-height: 1.6;
color: var(--color-dark-gray);

P: This is the standard paragraph text used for all main content. It should be highly legible and easy to read across all devices, ensuring a comfortable user experience.
